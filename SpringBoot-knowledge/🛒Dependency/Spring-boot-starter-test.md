# 1.spring-boot-starter-test
    Spring Boot 애플리케이션에서 단위 테스트, 통합 테스트, 웹 애플리케이션 테스트 
    등을 쉽게 작성하고 실행할 수 있도록 다양한 테스트 관련 라이브러리와 유틸리티를 
    포함한 스타터입니다.

# 2.포함된 라이브러리
## 2-1.JUnit 5
    자바 테스트 프레임워크의 표준으로, 테스트 작성과 실행을 위한 기본 
    라이브러리입니다.

## 2-2.Spring Test & Spring Boot Test
    스프링 컨텍스트를 로드하여 통합 테스트를 수행할 수 있도록 지원합니다.

## 2-3.AssertJ
    Fluent 스타일의 단언문을 제공하는 라이브러리로, 가독성 높은 테스트 코드를 작성할 
    수 있습니다.

## 2-4.Hamcrest
    매처(matchers)를 사용하여 유연한 단언문을 작성할 수 있습니다.

## 2-5.Mockito
    목 객체(mock objects)를 사용하여 단위 테스트를 수행할 수 있습니다.

## 2-6.JSONassert
    JSON 데이터를 비교하고 검증할 수 있는 라이브러리입니다.

## 2-7.JsonPath
    JSON 데이터를 탐색하고 추출할 수 있는 라이브러리입니다.
